# 🎓 AI-Powered Student Placement Prediction System

An intelligent Machine Learning-based placement prediction platform that estimates a student's placement probability using academic performance, skills, internships, and experience.

## 🚀 Features

- Placement Probability Prediction
- Interactive Dashboard
- Real-Time Analysis
- Career Recommendations
- Confidence Score Visualization
- Machine Learning Powered

## 🛠️ Technology Stack

- Python
- Streamlit
- Scikit-Learn
- Pandas
- NumPy

## 📂 Project Structure

app.py
placement.py
dataset.csv
model.pkl
accuracy.pkl
requirements.txt

## ▶️ Run Project

pip install -r requirements.txt

streamlit run app.py

## � Environment Configuration

Create a `.env` file in the project folder with:

```text
EMAIL_ADDRESS=yourgmail@gmail.com
EMAIL_PASSWORD=your_gmail_app_password
```

Use an App Password if you have 2-Step Verification enabled for your Gmail account.

Optionally set your app URL (used in email links):

```text
APP_URL=http://localhost:8501
```
